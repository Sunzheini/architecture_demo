# Prompt:
I will be team lead in a company that will develop a new software product, we don't have software 
engineers yet and no project software, repository, anything. We have to make them on the basis of 
best practices. First now we search for an architect to make an architecture of the software based 
on our requirements, that we have. The requirements are comprised of what modules we want (18 in 
total) and what each module will do. We want to use AI agents in each module where applicable. Now 
I will ask you questions, and you will answer it to me as concise as possible. What must be the 
output of the architect's work.

# Architect output:
1. High-Level System Design: Diagram and description of overall structure, showing how the 18 modules interact.
2. Technology Stack Recommendation: Specific languages, frameworks, and databases justified by your requirements.
3. AI Integration Strategy: A blueprint for where and how AI agents will be embedded in each applicable module (e.g., agent orchestration, model hosting, data flow).
4. Data Architecture: ER diagrams and specifications for data storage and flow between modules.
5. Infrastructure Design: Cloud vs. on-premise decision, deployment diagrams, and scalability plans.
6. API Contract / Interface Definitions: Clear specifications for how modules communicate (REST, GraphQL, message queues).
7. Technical Roadmap: Prioritized breakdown of which modules to build first based on technical dependencies.
